# Doriane Care ecommerce common repository

This repository will be used as a submodule of the other repositories for the [@jesuph-jobs/doriane-care](https://github.com/orgs/Jesuph-jobs/teams/doriane-care) project
it will be contain the following:
- `lib` directory for the libraries that will be loaded from the different front repositories
- `models` directory for the models that will be loaded from the different back repositories
- `validations` directory for the validations that will be loaded from all the different repositories
- `types` directory for the types that will be loaded from the different     repositories
